Sobre o Projeto

Esta é uma aplicação web desenvolvida com Flask, um microframework Python para desenvolvimento web rápido e simples.

A aplicação demonstra o conceito de processamento no servidor com download no cliente, onde:

  • O cliente (navegador) envia dados para o servidor
  • O servidor (Flask) processa os dados (converte texto para maiúsculas)
  • O resultado é retornado ao cliente
  • O cliente pode fazer download do resultado processado

Como Executar

  1. Instale o Flask: pip install flask
  2. Execute a aplicação: python app.py
  3. Acesse no navegador: http://localhost:5000
  4. Digite um texto, processe e faça o download do resultado!
